Mi glosario personal de Sistemas Operativos

Por: Juan Daniel López Torres



A continuación encontrarás las definiciones de las principales palabras y expresiones relacionadas con la ingeniería de sistemas

IP

Significa Protocolo de internet



SMTP

Significa protocolo simple de transferencia de correo



Protocolo

Conjunto de reglas que rigen la comunicación entre dispositivos en una red.



Topología

Estructura física o lógica de una red de computadoras.



Router

Dispositivo de red que dirige el tráfico de datos entre redes o subredes.



Firewall

Sistema de seguridad que controla el tráfico de red basado en reglas predefinidas.



Multiprocesamiento

La capacidad de un sistema operativo para aprovechar múltiples procesadores o núcleos de CPU.



Switch

Dispositivo de red que conecta dispositivos en una red local y gestiona el tráfico de datos.



LAN

Red de área local, que cubre un área geográfica limitada como una oficina o un edificio.



Sistema Operativo

Un software que gestiona los recursos de hardware y proporciona servicios comunes para los programas de aplicación.



VPN

Red privada virtual, que permite a los usuarios acceder a una red privada a través de una red pública.



Subred

División lógica de una red IP en segmentos más pequeños para gestionar el tráfico.



Ping

Herramienta utilizada para probar la conectividad de red enviando paquetes de datos a un host.



Sistema de Archivos

La estructura que el sistema operativo utiliza para organizar y almacenar archivos en un dispositivo de almacenamiento.



HTTP

Protocolo de transferencia de hipertexto, utilizado para la comunicación entre navegadores web y servidores.



MAC

Dirección de control de acceso a medios, que identifica de manera única a un dispositivo en una red.



Administrador de Dispositivos

Parte del sistema operativo que controla la comunicación entre el software y los dispositivos de hardware.



Ethernet

Estándar de comunicación de red por cable más comúnmente utilizado.



Virtualización

La capacidad de un sistema operativo para crear entornos virtuales que simulan hardware y software reales.



NAT

Traducción de direcciones de red, que permite que varios dispositivos compartan una única dirección IP pública.



DNS

Sistema de nombres de dominio



WAN

red de área extensa (Wide Area Network)
